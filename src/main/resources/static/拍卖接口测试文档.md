# 拍卖接口测试文档

## 接口信息
- **接口地址**: `POST /api/auction/bid`
- **认证方式**: Token认证
- **功能**: 用户参与拍卖出价

## 测试用例

### 1. 正常出价

**请求参数**:
```json
{
  "auction_id": "AUCTION001",
  "artwork_id": "ART001",
  "price": 1900000,
  "currency": "CNY"
}
```

**预期响应**:
```json
{
  "code": 1,
  "msg": "出价成功",
  "data": {
    "auction_id": "AUCTION001",
    "artwork_id": "ART001",
    "bid_id": "BID001",
    "user_id": "USER001",
    "price": 1900000,
    "bid_time": "2025-03-16 10:15:30",
    "is_current_winner": true,
    "current_price": 1900000,
    "next_min_price": 2000000
  }
}
```
